\section{Luglio 2017}

\subsection{Turno 1}

\subsubsection{Testo}

Si consideri il tipo di dato \sml{FOR = For of int * (int -> int);} i cui valori \sml{For(n, f)} rappresentano funzioni che implementano un ciclo for come il seguente:
\begin{lstlisting}[language = C, frame = none]
int ciclofor (int x) {
	for (int i = 0; i < n; i++) {
		x = f(x);
	}
}
\end{lstlisting}

Si scriva una funzione \sml{eval} (avente tipo \sml{FOR -> (int -> int)}) che riceve come argomento un valore di tipo \sml{FOR} e ritorna una funzione da interi ad interi che implementa il ciclo indicato qui sopra (applica \sml{n} volte la funzione \sml{f} all'argomento).

\medskip
Come esempio, se \sml{val f = fn x => x * 2}, allora \sml{eval (For(3, f))} ritornerà una funzione che dato un numero \sml{i} ritorna \sml{i * 8}:
\begin{lstlisting}[style = SML, frame = none]
> val f = fn x => x * 2;
val f = fn: int -> int

> eval (For(3, f));
val it = fn: int -> int

> val g = eval (For(3, f));
val g = fn: int -> int

> g 5;
val it = 40: int
\end{lstlisting}

\subsubsection{Soluzione}

\begin{lstlisting}[style = SML, caption = {Definizione della funzione \sml{eval}}]
datatype FOR = For of int * (int -> int);

val rec eval = fn For (n, f) =>
		fn x => if (n > 0) then
					eval (For (n - 1, f)) (f x)
				else
					x;
\end{lstlisting}
